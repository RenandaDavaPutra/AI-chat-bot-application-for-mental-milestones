# AI-chat-bot-application-for-mental-milestones
Aplikasi berbasis CLI (Command Line Interface) yang ditulis menggunakan bahasa Go (Golang) untuk membantu pengguna mengelola tenggat waktu tugas (milestone) sekaligus memonitor kesehatan mental. Aplikasi ini dikembangkan sebagai Tugas Besar untuk mengimplementasikan konsep struct, array, searching, dan sorting.
